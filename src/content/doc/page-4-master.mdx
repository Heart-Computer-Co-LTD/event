商品や得意先などのマスタ情報を登録する方法です。 ここでは商品マスタを例にして説明しますが、どのマスタも基本的な操作は同様です。

## 画面遷移図(リスト画面から)
リスト画面からの画面遷移図です。参照・登録・修正・複写の各画面に遷移できます。
```plantuml
@startuml
skinparam monochrome false

title 画面遷移図(リスト画面から)


state リスト画面
state 登録画面
state 参照画面
state 修正画面
state 複写画面

[*] --> リスト画面: メニュー選択
リスト画面 --> 参照画面: 参照（F1）
リスト画面 --> 登録画面: 登録（F2）
リスト画面 --> 修正画面: 修正（F3）
リスト画面 --> 複写画面: 複写（F12）
参照画面 --> 削除確認: 削除（F4）
@enduml
```
### リスト画面
登録済みのマスタ情報を一覧表示します。 一覧の表示項目は、各マスタによって異なります。CSVやExcelに出力することができます。
![image](https://github.com/Heart-Computer-Co-LTD/yui-manual/assets/22786124/492cd860-59b1-445e-862c-50bbfc70973a)

### 参照画面
登録済みのマスタ情報を参照します。 参照画面は、登録画面と同じ画面です。修正ができないようになっています。
前後や先頭末尾のレコードを参照することができます。削除ができるようになっています。
![image](https://github.com/Heart-Computer-Co-LTD/yui-manual/assets/22786124/ca913a09-5e4a-464e-872a-6d236fc65517)

### 登録画面
新規にマスタ情報を登録します。 登録画面は、参照画面と同じ画面ですが、削除ができないようになっています。
![image](https://github.com/Heart-Computer-Co-LTD/yui-manual/assets/22786124/3803a08b-8756-4090-b1f8-4170b599df0d)

### 修正画面
登録済みのマスタ情報を修正します。 修正画面は、参照画面と同じ画面ですが、削除ができないようになっています。
![image](https://github.com/Heart-Computer-Co-LTD/yui-manual/assets/22786124/5ea249ce-b989-4321-9a37-5eaab8001ff0)

### 複写画面
登録済みのマスタ情報を複写し新規に登録します。 複写画面は、参照画面と同じ画面ですが、削除ができないようになっています。
![image](https://github.com/Heart-Computer-Co-LTD/yui-manual/assets/22786124/37fb5afb-2000-4777-b276-46514da07486)

## 画面遷移図(参照画面から)
参照画面からの画面遷移図です。登録・修正・複写・削除の各画面に遷移できます。

```plantuml
@startuml
skinparam monochrome false

title 画面遷移図(参照画面から)


state リスト画面
state 登録画面
state 参照画面
state 修正画面
state 複写画面

リスト画面 --> 参照画面: 参照（F1）
参照画面 --> 登録画面: 登録（F2）
参照画面 --> 修正画面: 修正（F3）
参照画面 --> 複写画面: 複写（F12）
参照画面 --> 削除確認: 削除（F4）
@enduml
```

## 画面遷移図（連続登録）
登録画面からの画面遷移図です。確定後に連続登録ができます。
修正や複写、削除の場合も同様です。
```plantuml
@startuml
skinparam monochrome false

title 画面遷移図(登録画面から)


state リスト画面
state 登録画面


リスト画面 --> 登録画面: 登録（F2）
登録画面 --> 登録画面: 登録を続けますか？>OK
@enduml
```

## コード検索
登録や修正の際に、コードを入力する場合は、コード検索を利用することができます。
コード検索は、コードの入力欄にカーソルを合わせて、F5キーを押すことで表示されます。
